API for State abbreviations

This API can get you the abbreviations of all 50 US states in one call. Or search for a specific state. All the information is returned as a string dictionary, therefore no application logic needed. You can quickly try out this API by going to Swagger UI or the ReDoc UI.

This API is built using FastAPI. It's a Python micro webframework that is great for Machine Learning models. This is a simple API that was thrown together in a couple of days. This framework can do a lot more and is a go-to for handling Machine learning models. You can view the minimal code that powers this site and API in the GitHub repo.

This is a free API hosted on Deta.sh. They offer free hosting!
