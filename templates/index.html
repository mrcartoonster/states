API for State abbreviations

Welcome to US States abbreviations. This is a simple API that outputs US State abbreviations. There aren't many API's out there that simply output state abbreviations so this API now exists! You can just get a list of all the state abbreviations or you can search for the specific state you need. You can go view and try out this API from the browser by going to the Swagger UI or the ReDoc UI.

This is a free API hosted on Deta.sh using the explorer tier. This means that only 25K requests can be made to this API a month. I haven't rate limited it yet, but will if this API get's more then 25K requests a month. If you really like this service and want to up the requests per-month, sponsor this project on GitHub Sponsors and I'll add API keys so you can up the requests by moving this up a tier in Deta.sh.

This API is built using FastAPI. It's Python micro webframework and it's mission statment is FastAPI framework, high performance, easy to learn, fast to code, ready for production it definitely lives up to that mission statement. You can view the code on GitHub here. This is simple API that was thrown together in a couple of days. This framework can do a lot more and is a go to for handling Machine learning models. I went through this course from TestDriven.io that shows all the capabilites of this wonderful framework.
